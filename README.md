# Конвертер цветовых моделей XYZ ↔ Lab ↔ HLS

Описание
Веб-приложение для интерактивного преобразования цветов между профессиональными цветовыми моделями, используемыми в полиграфии, дизайне и компьютерной графике. Приложение позволяет работать с моделями XYZ (D65), CIELab (D65) и HLS, изменять значения компонентов с помощью ползунков и числовых полей, а также визуально наблюдать результат на цветовой панели. Встроенная палитра позволяет быстро выбирать готовые цвета и анализировать их представление в разных цветовых пространствах.

Возможности
* Поддержка трёх профессиональных цветовых моделей:
    * XYZ (D65) - аппаратно-независимая модель, основа цветовых преобразований
    * CIELab (D65) - равноконтрастное пространство, близкое к человеческому восприятию
    * HLS - интуитивная модель для работы с оттенками
* Двусторонняя синхронизация значений между моделями (изменение в одной модели автоматически обновляет остальные)
* Два режима ввода: ползунки для быстрой настройки и числовые поля для точного ввода значений
* Контроль цветового охвата с предупреждением о выходе за пределы sRGB гаммы
* Визуализация результатов:
    * Большая панель предпросмотра текущего цвета
    * Интерактивный цветовой круг для быстрого выбора
    * Палитра из 200 готовых оттенков
* Профессиональные функции:
    * Коррекция гаммы sRGB (2.4) при преобразованиях
    * Учет белой точки D65 (X=95.047, Y=100, Z=108.883)
    * Точные матричные преобразования между моделями
* Полностью автономное веб-приложение, работающее прямо в браузере без сервера
* Адаптивный дизайн для корректного отображения на компьютерах и мобильных устройствах

Технологии

Математический аппарат
Преобразования sRGB ↔ XYZ (D65)

// Гамма-коррекция sRGB
srgbToLinear(u) = (u ≤ 0.04045) ? u/12.92 : ((u+0.055)/1.055)^2.4

// Матрица преобразования RGB → XYZ
X = (0.4124564·R + 0.3575761·G + 0.1804375·B) × 100
Y = (0.2126729·R + 0.7151522·G + 0.0721750·B) × 100  
Z = (0.0193339·R + 0.1191920·G + 0.9503041·B) × 100


Преобразования XYZ ↔ CIELab (D65)

// Нормализация относительно белой точки D65
f(t) = (t > 0.008856) ? t^(1/3) : (7.787·t + 16/116)

L = 116·f(Y/Yn) - 16
a = 500·[f(X/Xn) - f(Y/Yn)]
b = 200·[f(Y/Yn) - f(Z/Zn)]


Преобразования RGB ↔ HLS

// Классические формулы из компьютерной графики
H = hue ∈ [0°, 360°]
L = lightness ∈ [0%, 100%]  
S = saturation ∈ [0%, 100%]



Как это работает
1. Выбор исходной модели Пользователь изменяет значения компонентов (ползунками или числами) в одной из моделей — XYZ, Lab или HLS
2. Каскадная конвертация Приложение выполняет точные математические преобразования между моделями:
    * XYZ ↔ Lab (через нормализацию D65)
    * XYZ ↔ RGB (через матрицы и гамма-коррекцию)
    * RGB ↔ HLS (через цилиндрические координаты)
3. Визуализация и контроль
    * Цвет отображается в блоке предпросмотра
    * Система предупреждает о выходе за пределы sRGB гаммы
    * Все модели автоматически синхронизируются
4. Работа с палитрой Дополнительная палитра из 200 цветов (10×20) позволяет быстро выбирать оттенки и анализировать их представление в разных моделях

Пример использования
1. Открыть приложение в современном браузере
2. Исследовать цвет #4EA6FF (исходный синий):
    * В XYZ: X=24.79, Y=25.37, Z=87.35
    * В Lab: L=66.57, a=10.87, b=-43.40
    * В HLS: H=208.24°, L=65.29%, S=100%
3. Изменить компонент L в Lab на 80 и наблюдать автоматическое обновление XYZ и HLS
4. Выбрать оранжевый цвет из палитры и проанализировать его координаты во всех моделях
5. Проверить граничные случаи — крайние значения в Lab, которые могут выходить за sRGB гамму
